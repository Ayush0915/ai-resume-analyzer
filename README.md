# 🚀 AI-Powered Resume Analyzer

An AI-based web application that analyzes resumes against job descriptions using NLP and BERT embeddings.

---

## 🔥 Features

- 📄 Resume parsing (PDF & DOCX)
- 🧠 Skill extraction engine
- 🤖 Semantic similarity using Sentence Transformers (BERT)
- ❌ Missing skill detection
- 💡 Intelligent feedback generation
- 🌐 Interactive Streamlit web app

---

## 🛠️ Tech Stack

- Python
- Streamlit
- scikit-learn
- Sentence Transformers
- Pandas
- PyPDF2
- python-docx

---

git clone https://github.com/Ayush0915/ai-resume-analyzer.git
